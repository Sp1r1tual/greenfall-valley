## Greenfall Valley (experimental)

An experimental isometric farming sandbox built to explore PixiJS rendering, pathfinding, and game loop architecture. Not a finished game – just a playground for ideas.

## About the Project

A small isometric world where you can place beds, trees, and barns, grow crops, and watch a character autonomously walk around and tend to the farm. Built as a personal experiment – no gameplay loop, no goals, just tinkering.

---

## Tech Stack

### Frontend

- TypeScript
- React
- PixiJS
